## Run Locally


Clone the project

```bash
  git clone [Project-link]
```

Go to the project directory

```bash
  cd [project-directory]
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Start the server

```bash
  streamlit run main.py
```
